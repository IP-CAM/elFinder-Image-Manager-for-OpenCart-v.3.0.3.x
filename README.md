# FileManager elFinder for OpenCart

Файловый менеджер и менеджер изображений для opencart.

## Описание

Добавляет возможность использования elfinder в опенкарт в редакторе summernote.

Языки:
- English
- Русский


### Совместимость
- OpenCart 3.0.3.x
